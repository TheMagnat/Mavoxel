## Getting Started

### Prerequisites

To compile this project, you need have installed this library :

-[GLFW](https://www.glfw.org/)
-[Glad](https://glad.dav1d.de/)

### Installing

#### Linux and Macos

Just run make

```
make
```


## Run

Execute the executable make created to start the exemple file
